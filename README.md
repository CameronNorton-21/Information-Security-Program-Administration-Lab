# Information Security Program Administration Lab

## Role Simulated
Information Security Program Analyst / Junior ISSO

---

## Overview

This project simulates the administration of an Information Security Management System (ISMS) for Sentinel Health Systems, a fictional healthcare software organization.

The objective of this project was to gain hands-on experience with:

- Information Security Governance
- Policy Management
- Risk Management
- Audit Readiness
- Compliance Reporting
- Security Documentation

---

## Company Profile

**Organization:** Sentinel Health Systems

**Industry:** Healthcare Software

**Employees:** 150

**Locations:** Baltimore, MD and Raleigh, NC

### Technology Stack

- Windows 11
- Microsoft 365
- Azure
- Active Directory
- SQL Server
- SharePoint
